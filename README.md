# Dotfiles

Dotfiles are all those `.` files that sit in your `~` and customize your system.
Here are mine.

## Usage

```bash
# clone the repository
cd $HOME
git clone https://github.com/atosatto/dotfiles.git .dotfiles
cd .dotfiles

# install everything
make
```
